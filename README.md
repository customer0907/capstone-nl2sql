# 서울시 공공데이터를 대상으로 한 도메인 특화 Text2SQL 모델 개발
- 2024-1 졸업 프로젝트(정보통신 종합설계)

## 프로젝트 소개
- (주)카카오의 Text2SQL 오픈소스 모델인 RYANSQL의 성능 지표를 개선하고,
자연어를 입력받아 SQL 쿼리를 출력하는 웹사이트를 구현

## 프로젝트 기간
- 24.02 - 24.06

## 학습 환경
1. OS
  - Linux : Ubuntu 20.04.5 LTS
2. HW
  - GPU : NVIDIA RTX A6000
  - RAM : 128GB
3. SW
  - python 3.8.10
  - pytorch 2.2.2
  - cuda 12.1
